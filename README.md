# inzynierka-parkour-ue5

Projekt do pracy inżynierskiej „Mechanika parkouru w grach wideo.

Środowisko
- Silnik: Unreal Engine 5.7.0
- Szablon: First Person Template (FPP)
- Platforma: PC (Windows)

Sterowanie
- WASD + mysz - ruch i rozglądanie
- E - Parkour (vault / mantle)
- L Ctrl - Slide

Uruchomienie
1. Otwórz plik `Inz_Projekt.uproject`
2. Uruchom projekt w UE 5.7
3. Kliknij Play (PIE)

Mapa testowa
- `Lvl_FirstPerson`

Gdzie jest logika
- `BP_PlayerCharacter`
- `BP_ParkourComponent`
